# Bootstrap 8 : Collapse et accordeon

Bootstrap fourni une possibilité de créer des éléments déroulants simplement :
les collapes.

En utilisant la documentation, vous allez créer 3 mini-exercices :

Premier :
    Créez 3 éléments de 4 colonnes chacun avec un bouton au dessus(voir bootstrap8-1.png).Chaque bouton déroule/réenroule le texte présent dessous.

Deuxieme :
    Créez Accordéon de 3 éléments qui se dérouleront au clic(voir bootstap-2.png).

Troisieme :
    Mixez les 2 exercices précédents : La présentation ressemblera au premier exercice mais lorsque l'un des boutons sera cliqué, les autres éléments se replieront s'ils sont dépliés.(un seul élément pourra être déplié à la fois)  (data-parent est votre ami)
