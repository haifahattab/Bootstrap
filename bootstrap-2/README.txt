# Bootstrap 2 : Deja vu

Grace à la grille de bootstrap, vous pouvez positionner les éléments HTML sans écrire une seule ligne de CSS.
Dans cet exercice, tentez de reproduire la configuration de l'image bootstrap2.png sur les formats medium et supérieurs 
et celle de l'image bootstrap2-portable.png sur les configurations plus petites.Les couleurs sont celles incluses dans bootstrap.

Indice : Commencez par écrire votre HTML dans l'ordre correspondant à l'image de portable, bootstrap empile automatiquement 
les éléments lorsque l'écran est petit.
