Rendez-vous sur la page de bootstrap (actuellement v4.3.1) (Lien vers page de téléchargement) et regardez la section BootstrapCDN. Il vous propose 2 balises à copier pour bootstrap (css et js) et 2 balises qui concernent jquery et popper. Copiez ces balises et inserez les dans le HEAD.
Cependant, pour utiliser proprement les fonctionnalités de bootstrap(de son JS surtout), vous devrez placer le script qui pointe vers bootstrap.min.js après les script de popper et jquery.
Vous devriez avoir dans l'ordre : le lien vers bootstrap.min.css puis le script de jquery puis le script popper puis bootstrap.min.js . Quand vous souhaiterez ajouter votre propre CSS ou JS, Vous les placerez sous ces balises.
Rendez-vous sur la page de bootstrap (actuellement v4.3.1) (Lien vers page de téléchargement) et regardez la section Compiled CSS & JS. Il vous propose de télécharger un zip contenant les différents éléments. Ceux qui vous interessent ici sont : bootstrap.min.css et bootstrap.min.js . Extrayez ces 2 fichiers dans le dossier contanant votre projet et référencez les dans index.html comme d'habitude
ATTENTION : Certaines fonctionnalités de bootstrap nécessitent jquery et popper qui ne sont pas inclus dans le zip, vous devrez les télécharger à part et y faire référence dans index.html

Une astuce pour vous faciliter la vie avec votre IDE : Faites vous un snippet générique d'une page avec bootstrap.
Avec ATOM :
fichier puis snippets puis collez le code suivant et sauvegardez.Lorsque vous taperez boot suivi de tab dans un fichier html, ce code sera écrit. '.source.html': 'Bootstrap': 'prefix': 'boot' 'body': """ <!DOCTYPE html> <html lang="fr"> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width, initial-scale=1"> <title>Mon titre</title> <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.0/css/bootstrap.min.css"> <script src="https://code.jquery.com/jquery-3.3.1.slim.min.js"></script> <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.7/umd/popper.min.js"></script> <script src="https://stackpath.bootstrapcdn.com/bootstrap/4.3.0/js/bootstrap.min.js"></script> <link rel="stylesheet" type="text/css" href="style.css"> <script type="text/javascript" src="script.js"></script> </head> <body> $1 </body> </html> """
