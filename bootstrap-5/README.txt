# Bootstrap 5 : les navbars

Pour cet exercice, vous aller devoir créer 3 navbars :
- une navbar horizontale : celle ci contiendra 3 éléments centrés (Accueil, a propos et nous contacter).
- une navbar verticale : Celle ci fera 2 colonnes de large(par rapport à la page) avec les mêmes éléments que précédemment
- une navbar verticale avec contenu : cette navbar permet d'afficher différents éléments textes (regardez sur la documentation au niveau des navs dans le chapitre javascript behavior)

Vous avez un exemple de rendu sur l'image bootstrap5.png .
